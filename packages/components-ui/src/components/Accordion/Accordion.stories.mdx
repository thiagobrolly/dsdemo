# Accordion

O componente `Accordion` permite que o usuário exiba e oculte uma seção de conteúdo associada.

## Import

Importe todas as partes.

```sh
  import { Accordion, AccordionItem, AccordionTrigger, AccordionContent } from '@dsdemo/components';
```
