# Modal

O componente `Modal` é um tipo de janela que aparece na frente do conteúdo do aplicativo para fornecer informações críticas ou solicitar uma decisão. As caixas de diálogo desativam toda a funcionalidade do aplicativo quando elas são exibidas e permanecem na tela até que sejam confirmadas, rejeitadas ou que uma ação necessária tenha sido executada.

## Import

```sh
  import { Modal } from '@dsdemo/components';
```

## Customized

O `Modal` pode ser customizado através das props:

`shouldCloseOnOverlayClick` para ativar ou desativar a opção de fechar ao clicar fora do modal.

`shouldCloseOnEscClick` para ativar ou desativar a opção de fechar ao pressionar a tecla Esc.

`scrollBarDisabled` para ativar ou desativar o scroll do conteúdo por trás do modal.

`viewCloseButton` exibir botão para fechar modal.

`title` para adicionar um título ao conteúdo o modal.

`widthModal` para informar um tamanho para o modal.
